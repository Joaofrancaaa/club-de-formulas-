🧪 Acervo Digital de Fórmulas — Naturalis
Plataforma de Sales Enablement desenvolvida para acelerar prescrições médicas e aumentar a conversão comercial de fórmulas manipuladas.

🚀 Visão Geral
O Acervo Digital de Fórmulas resolve gargalos reais do setor de farmácias de manipulação:

Problema: Médicos gastam tempo excessivo formatando prescrições e representantes perdem o timing da venda em ferramentas não integradas (WhatsApp/Instagram).

Solução: Uma plataforma de fricção zero que centraliza fórmulas validadas, permitindo cópia imediata e solicitação de orçamento direta via API do WhatsApp.

✨ Principais Funcionalidades
📄 Lâminas Técnicas Digitais: Conteúdo focado em conversão com embasamento científico e mecanismos de ação.

⚡ One-Click Copy: Sistema de cópia inteligente que já formata a fórmula para o padrão de prontuários médicos.

💬 Integração com WhatsApp: Geração automática de templates de mensagens para o representante comercial.

🧠 Organização Inteligente: Segmentação por categorias (Humana, Veterinária, etc.).

🔐 Painel Administrativo (CMS): Interface completa para gestão de conteúdo (CRUD) com validação de dados.

🔗 Link na Bio (/bio): Página otimizada para redes sociais que mantém o tráfego dentro do domínio principal.

🛠️ Stack Tecnológica
Framework: Next.js 14+ (App Router)

Linguagem: TypeScript

Banco de Dados & ORM: Prisma + Supabase

Formulários: React Hook Form + Zod

Estilização: CSS Modules

Icons: Lucide React

🧠 Arquitetura & Decisões de Projeto
O projeto foi construído focando em Performance e Escalabilidade:

Server Components: Utilizados para busca de dados e SEO, garantindo carregamento quase instantâneo.

Segurança: Proteção de rotas administrativas e limites de login.

UX de Conversão: Diferente de um CRUD comum, a interface foi desenhada para reduzir cliques e acelerar o fluxo comercial.
